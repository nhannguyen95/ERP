### Input

`data(i, j)` is the rating that user _ith_ rated for movie _jth_.

`data(i, j) = -1` means unrated.

```
5,3,-1,1
4,-1,-1,1
1,1,-1,5
1,-1,-1,4
-1,1,5,4
```

### Output

Try to predict the users' rating where `data(i, j) = -1`.
